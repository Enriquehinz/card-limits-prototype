# Deployment Guide - Share Your Prototype

Your prototype is now on GitHub! Here are several ways to share it with others via a URL:

## Option 1: GitHub Pages (Free & Easy) ‚≠ê Recommended

### Steps:

1. Go to your repository on GitHub: `https://github.com/Enriquehinz/card-limits-prototype`

2. Click on **"Settings"** (top menu bar)

3. Scroll down to **"Pages"** in the left sidebar

4. Under **"Source"**, select:
   - Branch: `main`
   - Folder: `/ (root)`
   
5. Click **"Save"**

6. Wait 1-2 minutes, then your site will be live at:
   ```
   https://Enriquehinz.github.io/card-limits-prototype/
   ```

### Note:
- It may take a few minutes for the site to go live
- Any updates you push to the `main` branch will automatically update the live site
- The URL will be: `https://YOUR_USERNAME.github.io/card-limits-prototype/`

---

## Option 2: Netlify Drop (Super Quick)

1. Go to [netlify.com/drop](https://app.netlify.com/drop)
2. Drag and drop your `index.html` file (or the entire folder)
3. Get an instant URL like: `https://random-name-123.netlify.app`

---

## Option 3: Vercel (Also Quick)

1. Go to [vercel.com](https://vercel.com)
2. Sign up/login with GitHub
3. Click "New Project"
4. Import your GitHub repository
5. Deploy - get a URL like: `https://card-limits-prototype.vercel.app`

---

## Option 4: Share GitHub Repository Link

You can also share the repository link directly:
```
https://github.com/Enriquehinz/card-limits-prototype
```

People can:
- View the code
- Download/clone the repository
- Open `index.html` locally
